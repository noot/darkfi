Expand description

The Sparse Merkle Tree struct.

SMT stores a set of leaves represented in a map and a set of empty hashes that it uses to represent the sparse areas of the tree.

The trait param N is the depth of the tree. A tree with a depth of N will have N + 1 levels.

Fields§

§store: S

A map from leaf indices to leaf data stored as field elements.

§hasher: H

The hasher used to build the Merkle tree.

§empty_nodes: &'a [F; M]

An array of empty hashes hashed with themselves N times.
